## 🔧 技术配置

### 代码中的配置

```typescript
// src/config/credits.ts
export const CREDIT_CONFIG = {
  VIDEO_GENERATION: {
    BASE_COST: 15,        // 5秒视频消耗15积分
    IMAGE_COST: 5,        // 每张参考图片+5积分
    DURATION_MULTIPLIER: {
      5: 1,               // 5秒 × 1
      10: 2,              // 10秒 × 2 = 30积分
    }
  }
}
```
